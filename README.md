# 📊 Exploratory Data Analysis (EDA) Projects

This repository contains multiple **Exploratory Data Analysis (EDA)** projects performed on different real-world datasets using Python. The aim is to understand data, uncover patterns, and generate meaningful insights through visualization and statistical techniques.

---

## 🚀 Project Overview

This repository includes EDA performed on various datasets such as:

* 🏠 House Price Prediction Dataset
* 🚢 Titanic Dataset
* 🏦 Bank Loan Dataset
* 🏅 Olympics Historical Dataset
* 🏥 Healthcare Dataset

Each notebook focuses on extracting insights using structured analysis and visualization techniques.

---

## 🎯 Objectives

* Understand dataset structure and features
* Perform data cleaning and preprocessing
* Analyze relationships between variables
* Visualize patterns and trends
* Prepare data for machine learning

---

## 🛠️ Tools & Libraries Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook / Google Colab
